### 🛫 **Project Title**: Flight Management System  
### 🧠 **Domain**: Data Structures and Algorithms (DSA)  
### 💻 **Language Used**: Python  

### 📄 **Project Description**:

The **Flight Management System** is a mini project designed to simulate and manage flight-related operations using core Data Structures and Algorithms in Python. The system allows users (such as airport staff or travel agents) to manage and track flight details efficiently.

This project demonstrates the practical use of **DSA concepts** such as:

- **Linked Lists** – To manage the list of flights (insertion/deletion of flights).
- **Queues** – For managing the passenger check-in and boarding process.
- **Stacks** – For storing recent flight searches or navigation history.
- **Binary Search Trees (BST)** – For searching and sorting flight information like departure time, flight ID, or destination.
- **Graphs** – (Optional) To represent routes between different airports.

---

### 🧩 **Key Functionalities**:

1. **Add a Flight** – Insert new flight data using linked list or BST.
2. **Delete a Flight** – Remove a flight from the system using list operations.
3. **Search Flights** – Use binary search or BST for fast lookup.
4. **Sort Flights** – By departure time, destination, or price using sorting algorithms.
5. **Passenger Check-In Queue** – FIFO queue system for handling passengers.
6. **Flight History (Back feature)** – Stack implementation to go back to previous flight searches.
7. **Display All Flights** – Traverse and display flight information.
8. **Route Finder (Optional)** – Use graphs to find shortest paths between airports.

---

### 🎯 **Learning Outcomes**:

- Hands-on experience with data structures in real-life scenarios.
- Implementation of efficient searching, sorting, and data organization.
- Understanding the importance of choosing the right data structure for specific tasks.
